# Teammors 自述文件

安全、智能、私密、便捷——这是 Teammors 的目标。

Teammors 不仅是一款专业的即时通讯工具，更是一个人工智能知识库用户终端。

Teammors 目前支持以下平台：**macOS、Android、iOS、Windows 和 Linux**。

## Teammors 功能

### 🔒 高级安全与隐私保护

- **端到端加密 (E2EE)**：聊天（包括群聊）默认启用，仅双方可见，消息不会存储在服务器上。

- **阅后即焚消息**：可为私聊和群聊设置消息阅后即焚倒计时。

- **匿名性**：注册仅需输入一个字符串。

- **禁止数据挖掘**：不会进行任何数据收集。

### 🤖 可定制的智能AI机器人

- **第三方开发者**：用户可以接入多个知识库机器人，每个机器人拥有独立的频道，负责消息推送和接收。**5万+机器人集群**：每个机器人频道支持超过5万订阅用户。

**超级机器人频道**：稳定高效的机器人频道，提供机器人SDK，支持各种RAG集成。

### 💬 丰富的私聊和群聊功能

- 支持多种多媒体消息类型，包括表情符号、图片、语音消息和视频消息。

- 支持免费语音和视频通话。

- 支持大文件上传。

- 支持消息编辑和回复功能。

- 支持互相清理消息。

- 支持设置消息清理时间。

- 支持标记重要对话。

### 🎯 多媒体支持

- 支持免费语音通话和视频通话。

- 支持多种多媒体消息类型，包括表情符号、图片、语音消息和视频消息。

- 支持上传大文件，最大可达 2GB。

### 🌐 跨平台支持

- 支持多设备登录；同一账号可同时在多个设备上登录。

- 支持多种语言；用户可以随时切换语言；目前支持超过 20 种语言。支持多种平台客户端：macOS、Android、iOS、Windows、Linux

## 构建 Teammors 智能服务流程

1. **下载客户端**：从 https://www.teammors.top/ 下载相应的客户端

2. **注册账号**：注册一个账号

3. **创建机器人**：创建一个机器人

<img src="./images/robot-manage.png" width="700" alt="配置文件">

<img src="./images/add-robot.png" width="600" alt="创建机器人">

4. **配置机器人**：配置机器人的基本信息

<img <img src="./images/robot-info.png" width="600" alt="配置机器人">

5. **复制令牌**：复制机器人的令牌以备后用。

6. **下载示例项目**：下载本示例中的示例项目（Spring Boot 项目）。

7. **创建智能代理**：在第三方平台或自建智能代理服务平台上创建智能代理服务。

8. **获取凭证信息**：获取智能代理对应的访问 URL、访问令牌和其他信息。

9. **配置参数**：将获取的凭证信息填写到 Spring Boot 配置文件中。

<img src="./images/config-file.png" width="700" alt="配置文件">

10. **运行机器人**：启动 Spring Boot 项目。
